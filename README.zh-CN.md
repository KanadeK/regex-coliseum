# Regex Coliseum

通过正确率、性能和可读性三维评分学习 JavaScript 正则表达式的本地优先训练场。

它内置 30 道原创题，覆盖提取、校验、日志和重构；每次运行都在独立 Web Worker 中执行，250 ms 未返回就终止并重建。公开样例、隐藏样例和性能语料均随仓库提供，排行榜只存于当前浏览器。分享链接只包含题目 id，不会携带你的表达式或隐藏答案。

```bash
npm ci
npm run dev
```

在 Vite 给出的本地地址中输入不含 `/` 分隔符的正则源码，选择题目后点击“Run challenge”。

完整英文说明、架构、测试和安全边界请见 [README.md](README.md)。
